rex.rechunk_h5.rechunk_cli module¶

rechunk¶

RechunkH5 CLI entry point

rechunk [OPTIONS] COMMAND [ARGS]...

Options

-src, --src_h5 <src_h5>¶

Source .h5 file path [required]

-dst, --dst_h5 <dst_h5>¶

Destination path for rechunked .h5 file

--var_attrs <var_attrs>¶

.json containing variable attributes [required]

--version <version>¶

File version number

-m, --meta <meta>¶

Path to .csv or .npy file containing meta to load into rechunked .h5 file

-s, --process_size <process_size>¶

Size of each chunk to be processed

--log_file <log_file>¶

Path to .log file

-v, --verbose¶

If used upgrade logging to DEBUG

Rechunk h5 command line interface
