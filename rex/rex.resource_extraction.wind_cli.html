rex.resource_extraction.wind_cli module¶

WIND¶

WindX Command Line Interface

WIND [OPTIONS] COMMAND [ARGS]...

Options

-h5, --wind_h5 <wind_h5>¶

Path to Resource .h5 file [required]

-o, --out_dir <out_dir>¶

Directory to dump output files [required]

-t, --compute_tree¶

Flag to force the computation of the cKDTree

-v, --verbose¶

Flag to turn on debug logging. Default is not verbose.

multi-site¶

Extract multiple sites given in ‘–sites’ .csv or .json as “latitude”, “longitude” pairs OR “gid”s

WIND multi-site [OPTIONS] COMMAND [ARGS]...

Options

-s, --sites <sites>¶

.csv or .json file with columns “latitude”, “longitude” OR “gid” [required]

dataset¶

Extract given dataset for all sites

WIND multi-site dataset [OPTIONS]

Options

-d, --dataset <dataset>¶

Dataset to extract [required]

sam¶

Extract SAM variables

WIND multi-site sam [OPTIONS]

Options

-h, --hub_height <hub_height>¶

Hub height to extract SAM variables at [required]

region¶

Extract a single dataset for all pixels in the given region

WIND region [OPTIONS]

Options

-d, --dataset <dataset>¶

Dataset to extract [required]

-r, --region <region>¶

Region to extract [required]

-col, --region_col <region_col>¶

Meta column to search for region

sam-file¶

Extract all datasets at the given hub height needed for SAM for nearest pixel to the given (lat, lon) coordinates OR the given resource gid

WIND sam-file [OPTIONS]

Options

-h, --hub_height <hub_height>¶

Hub height to extract SAM variables at [required]

-ll, --lat_lon <lat_lon>¶

(lat, lon) coordinates of interest

-g, --gid <gid>¶

Resource gid of interest

site¶

Extract a single dataset for the nearest pixel to the given (lat, lon) coordinates OR the given resource gid

WIND site [OPTIONS]

Options

-d, --dataset <dataset>¶

Dataset to extract [required]

-g, --gid <gid>¶

Resource gid of interest

-ll, --lat_lon <lat_lon>¶

(lat, lon) coordinates of interest

timestep¶

Extract a single dataset for a single timestep Extract only pixels in region if given.

WIND timestep [OPTIONS]

Options

-d, --dataset <dataset>¶

Dataset to extract [required]

-ts, --timestep <timestep>¶

Timestep to extract [required]

-r, --region <region>¶

Region to extract

-col, --region_col <region_col>¶

Meta column to search for region
